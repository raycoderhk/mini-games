## 🛠️ 如何更新

### 方法 1：直接編輯 HTML

1. 打開 `repairs.html`
2. 搵到對應嘅分類（緊急/水喉/冷氣/電器）
3. 複製呢個模板：

```html
<div class="contact-card [category]" data-category="[category]">
    <span class="service-type [category]">[服務類型]</span>
    <h3 class="contact-name">[師傅名]</h3>
    <div class="contact-details">
        <div class="detail-item">
            <span class="detail-icon">📞</span>
            <span>[電話]</span>
        </div>
        <div class="detail-item">
            <span class="detail-icon">🛠️</span>
            <span>[專長]</span>
        </div>
    </div>
    <a href="tel:[電話]" class="call-button">📞 立即致電</a>
</div>
```

4. 替換 `[ ]` 入面嘅內容
5. Commit + Push

### 方法 2：Discord 分享

喺 Discord #community-resources Channel 分享，由負責人更新。
