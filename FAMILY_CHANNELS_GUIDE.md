## 🔧 手動創建步驟

### Step 1: 創建 Category
1. 喺 Discord Server 右鍵
2. 點擊 **"Create Category"**
3. 名稱：`Family` 或 `家庭`
4. 設置權限 (只限你同家人可見)

### Step 2: 創建三個頻道
在 Family Category 內創建：

| 頻道名稱 | 用途 |
|----------|------|
| `son-兒子` | 兒子相關事項 |
| `daughter-女兒` | 女兒相關事項 |
| `wife-太太` | 太太相關事項 |

### Step 3: 複製 Channel ID
1. Discord Settings → Advanced → 開啟 **Developer Mode**
2. 右鍵點擊每個新頻道
3. 點擊 **"Copy Channel ID"**
4. 話我知 ID，我即刻幫佢哋發送資料！
