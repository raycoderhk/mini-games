## 📋 **所有指令**

| 指令 | 說明 | 例子 |
|---|---|---|
| `!start [名字]` | 開始新遊戲 | `!start 大文` |
| `!play` | 繼續遊戲 | `!play` |
| `!status` | 查看狀態 | `!status` |
| `!save [槽位]` | 儲存進度 | `!save 1` |
| `!load [槽位]` | 讀取進度 | `!load 1` |
| `!help` | 顯示說明 | `!help` |

## 🎮 **遊戲流程**

```
1. !start 你的名字
   ↓
2. Bot 發送遊戲開始訊息
   ↓
3. 林娜出現，問第一個問題
   ↓
4. 輸入 1, 2, 或 3 選擇
   ↓
5. 對話繼續，關係值變化
   ↓
6. 到達結局
   ↓
7. 重新開始或探索其他分支
```

---

## 💡 **提示**

- **關係系統**: 你的選擇會影響林娜對你的態度
- **多結局**: 基於關係值，有 3 種不同結局
- **重複遊玩**: 每次選擇不同路徑，體驗不同故事
- **隨時查詢**: 用 `!status` 查看當前關係狀態

---

## 🐛 **故障排除**

### Bot 無回應？
1. 檢查 Bot 是否在 Server 中
2. 確認 Bot 有該頻道的發送訊息權限
3. 重啟 Bot: `node discord-bot.js`

### 指令無效？
- 確保使用 `!` 前綴 (不是 `！`)
- 指令不區分大小寫 (`!START` = `!start`)

### 遊戲進度丟失？
- 使用 `!save 1` 儲存進度
- 下次用 `!load 1` 讀取
