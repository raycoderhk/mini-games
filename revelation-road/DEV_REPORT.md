## 🎯 **遊戲流程預覽**

```
開始遊戲 → 輸入名字
    ↓
抵達启示路 (場景描述)
    ↓
遇見林娜 (接待員)
    ↓
對話選擇：
├─ 旅途順利嗎？
│  ├─ 還算順利 (+1 關係)
│  ├─ 比预期好 (+2 關係)
│  └─ 不想談 (-1 關係)
│
├─ 來启示路為了什麼？
│  ├─ 尋找答案 (+1 關係)
│  ├─ 重新開始 (+1 關係)
│  └─ 與你無關 (-2 關係)
│
└─ 禁止區域警告
   ├─ 追問消失事件 (+2 關係，獲得情報)
   └─ 簡單感謝 (+1 關係)

結局 (基於關係值):
├─ 關係 ≥ 2: 林娜對你有好感，願意分享情報
├─ 關係 0-1: 保持專業態度
└─ 關係 < 0: 對你有戒心
```
